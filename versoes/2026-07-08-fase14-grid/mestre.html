    🧑‍🌾 NPCs da Campanha

    NPCs persistem entre sessões (diferente dos avulsos do combate). Marque 👁️ para os jogadores os verem na aba "NPCs" deles — as notas privadas 🔒 nunca são enviadas. NPCs com bloco de combate podem entrar no rastreador pelo botão + NPC da aba Combate.

    💾 Meu Banco de NPCs

    Banco PESSOAL: os NPCs guardados aqui seguem você entre campanhas. Use 💾 no cartão de um NPC da campanha para o guardar, e 📥 para trazer um NPC do banco para a mesa ativa.

    📖 Minhas Aventuras

    Estruture a campanha como um livro-jogo: cada NÓ é uma cena (narração, encontro, social...) e cada SAÍDA é uma escolha possível do grupo. As aventuras ficam na sua biblioteca PESSOAL (seguem você entre campanhas); "▶ Iniciar" copia a aventura para a mesa ativa. Escolhas na tela dos jogadores chegam numa fase futura.
